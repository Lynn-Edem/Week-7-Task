# Personal Mood Dashboard

Single-page task using only DOM, events, and element `.style`.

Features:
- Live name greeting: "Hello, [Name]!" (defaults to "Hello, friend!")
- Grow / Shrink a mood box
- Toggle Day/Night mode
- Show / hide a secret motivational message
- Contains an in-code comment answering the difference between `.innerHTML` and `.innerText`

Files:
- index.html
- script.js
